Salt Minion 0.17.5-2 64-bit on Windows

This software asset installs the Salt Minion 64-bit on Windows by downloading from SaltStack.com, or by copying from the media folder if the installer is provided there. To use this asset to install another version, simply change the %INSTALLER% variable in the install-salt-minion.bat script. For CONS3RT deployments, this install script looks for either a Scenario role name or a Deployment runtime propety with the name "salt-master". If salt-master is found, the script automatically installs Salt Minion to point to the salt-master. Otherwise, it downloads or copies the Salt Minion installer to C:\cons3rt for manual installation after logging in. Note: Automatic installation may fail if you define BOTH a Scenario Role Name and a Deployment Runtime Property.

Overview

Salt is a new approach to infrastructure management. Easy enough to get running in minutes, scalable enough to manage tens of thousands of servers, and fast enough to communicate with them in seconds.  Salt delivers a dynamic communication bus for infrastructures that can be used for orchestration, remote execution, configuration management and much more.

Prerequisites

  • For automatic installation, either define a Scenario Role Name of salt-master, or a Deployment Runtime Property of salt-master.
  • DO NOT define BOTH a Scenario Role Name AND Deployment Runtime Property with the name: salt-master.
